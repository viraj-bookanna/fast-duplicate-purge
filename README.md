# fast-duplicate-purge
super fast duplicate line remover written in go

## prerequisites

install golang if not
```
sudo apt install golang
```

install requirements
```
go mod init main.go
go mod tidy
```


## Usage

```
go run main.go <input_file> <output_file>
```
